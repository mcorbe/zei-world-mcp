MCP Server

Zei World MCP

Explore ESG & CSR company data from Zei World directly inside ChatGPT, Claude, or any MCP-compatible client.

Endpoint

https://zei-world-mcp.vercel.app/mcp

Connect from ChatGPT

Requires ChatGPT Pro, Team, Enterprise, or Edu.

  1. Open ChatGPT and go to Settings → Apps
  2. Click New App (or the + button)
  3. Fill in:
    • Name: Zei World
    • Description: ESG/CSR company data from Zei World
    • MCP Server URL: paste the endpoint URL above
    • Authentication: No Auth
  4. Check "I understand and want to continue" and click Create
  5. In a new chat, click the connectors icon in the composer and enable Zei World

Connect from Claude Desktop

  1. Open your Claude Desktop config file:
    ~/.claude/claude_desktop_config.json (macOS/Linux) or
    %APPDATA%\Claude\claude_desktop_config.json (Windows)
  2. Add the following to "mcpServers":
    { "zei-world": { "type": "url", "url": "https://zei-world-mcp.vercel.app/mcp" } }
  3. Restart Claude Desktop
